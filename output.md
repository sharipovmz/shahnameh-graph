## **2. БАЛОИҲАГИРӢ ВА АМАЛИСОЗИИ БАРНОМА** {#балоиҳагирӣ-ва-амалисозии-барнома .unnumbered}

Дар доираи Ташкили система ва назорати давомоти толибилм дар мактаби
миёна технологияҳои зерин истифода мешаванд. Барои назорати версия ва
ҳамоҳангии кори гурӯҳӣ, истифодаи Github мувофиқ мебошад, ки таърихи
тағйирот ва рушдро сабт менамояд.

Қисмати Front-end-и система бо истифода аз HTML, CSS ва JavaScript таҳия
карда мешавад, ки имкон медиҳад интерфейси содда ва фаҳмо барои
омӯзгорон ва маъмурият эҷод карда шавад. Бо истифода аз HTML сохтори
саҳифаҳо, бо CSS ороиш ва мутобиқсозии намуди зоҳирӣ ва тавассути
JavaScript функсияҳо барои сабт, таҳрир ва нишон додани маълумоти ҳузур
таъмин карда мешавад.

Қисмати Backend бо истифода аз PHP таҳия карда мешавад, ки бо ёрии он
метавонем мантиқ, коркарди дархостҳо, пайвастшавӣ ба пойгоҳи додаҳо ва
идоракунии маълумотро таъмин намоем.

PHP дар якҷоягӣ бо MySQL барои нигоҳдорӣ ва коркарди маълумоти
толибилмон, синфҳо ва сабти ҳузур истифода мешавад. Илова бар ин,
механизмҳои иҷозатдиҳӣ ва аутентификатсия барои таъмини амнияти дастрасӣ
ба система низ бо истифода аз PHP амалӣ карда мешаванд.

### **Интихоби воситаҳои таҳия барои барнома**

Барои таҳияи қисмати сервер (Backend)-и системаи Ташкили система ва
назорати давомоти толибилм дар мактаби миёна технологияҳои муосир дар
асоси забони PHP интихоб гардиданд, ки барои сохтани меъмории боэътимод
ва мутобиқ барои ниёзҳои муассисаҳои таълимӣ мувофиқ мебошанд.

**2.1.1. Draw.io (diagrams.net)** ин як воситаи ройгони онлайн барои
сохтани диаграммаҳо мебошад, ки барои намоиши сохтори система, базаи
маълумот, равандҳои тиҷоратӣ ва дигар қарорҳои меъморӣ мувофиқ аст. Он
интерфейси фаҳмо ва осон барои истифода дорад, инчунин намудҳои гуногуни
диаграммаҳоро дастгирӣ мекунад, масалан, блок-схемҳо, UML,
ER-диаграммаҳо ва дигарҳо. Draw.io имкон медиҳад, ки бо истифода аз
шаблонҳо диаграммаҳо сохта шавад. Ғайр аз ин, он имконияти ҳамкории
гурӯҳӣ ва нигоҳ доштани диаграммаҳоро дар абр ва ҳамчунин истифодаи
офлайнро пешниҳод мекунад.

**2.1.2. Забони барномасозии PHP** ин яке аз забонҳои маъруф ва васеъ
истифодашаванда барои таҳияи барномаҳои вебӣ мебошад. Он барои эҷоди
вебсайтҳо ва системаҳои иттилоотӣ, аз ҷумла низоми назорати давомоти
толибилм дар мактаби миёна, хеле мувофиқ аст.

PHP имконият медиҳад, ки функсияҳои кор бо пойгоҳи додаҳо, сабти ҳузури
хонандагон, идоракунии маълумоти синфҳо ва таҳияи гузоришҳо осон ва
самаранок амалӣ карда шаванд.

Бо шарофати маҷмӯи модулҳои гуногун ва васлшавандаҳои омода, PHP
метавонад бо системаҳои аутентификатсия, идоракунии сессияҳо ва ҳамгироӣ
бо василаҳои паёмрасонӣ, аз ҷумла фиристодани огоҳиномаҳо тавассути
почтаи электронӣ, пайваст карда шавад. (SMS, почтаи электронӣ) \[14 с.
3\].

Соддагӣ, чандирӣ ва мувофиқат бо серверҳои вебии маъмул, PHP-ро ба яке
аз забонҳои мувофиқ барои ин намуди системаҳо табдил додааст.

**2.1.3. Платформаҳои PHP ва MySQL** инҳо ҳамчун якҷояи пурқувват барои
таҳияи системаҳои вебӣ, васеъ истифода мешаванд.

PHP платформаи чандир ва кроссплатформавӣ буда, барои сохтани барномаҳои
веб ва API-ҳо, ки муоширати байни қисми муштарӣ (Front-end) ва сервер
(Backend)-ро таъмин мекунанд, хеле мувофиқ аст.

Модулҳои асосие, ки бо истифода аз PHP таҳия мешаванд, инҳоянд:

-   Модули идоракунии корбарон (сабти ном, воридшавӣ, таъини нақшҳо);

-   Модули сабти ҳузури толибилмон ва идоракунии давомоти онҳо;

-   Модули таҳияи ҳисоботҳо ва гузоришҳои ҳузур барои маъмурият ва
    омӯзгорон;

-   Модули огоҳиномаҳо (фиристодани паёмҳо тавассути почтаи электронӣ ё
    дигар василаҳо оиди ҳузур ва сабти он) \[15 с. 78\].

Ин платформа бо шарофати мутобиқат бо серверҳои гуногун ва осонии
ҳамгироӣ бо маҷмӯи модулҳои иловагӣ, барои рушди чунин системаҳо хеле
муносиб мебошад.

**2.1.4. Муҳаррири код: VS Code** ин як муҳаррири код бисёр қавӣ ва
чандир аст, ки барои таҳияи барномаҳо дар PHP хеле мувофиқ аст. Он бо
функционалитети пурқувват, плагинҳо ва экс tenсияҳо барои таҳрир, таҳлил
ва санҷиши код бо осонӣ мутобиқ мегардад, ки имкон медиҳад таҳиягарон
барои навиштани код бо PHP ва дигар забонҳо истифода баранд. Қодириятҳои
интегратсия бо дигар системаҳо ва муҳити корӣ, ки барои PHP зарур аст,
инчунин кӯмак мекунад дар осон кардани раванди таҳия ва санҷиши
барномаҳо.

**2.1.5. Муҳаррири сервер: OpenServer** ин як муҳити серверии маҳаллӣ
аст, ки барои эҷод ва санҷиши барномаҳои веб дар PHP, MySQL ва дигар
серверҳои веб дар муҳити маҳаллӣ истифода мешавад. Ин муҳити пурқувват
имкон медиҳад, ки барномаҳои PHP дар муҳити локалӣ бе зарурати истифодаи
сервери веби ҳақиқӣ таҳия ва санҷида шаванд. OpenServer ҳамчунин якҷоя
бо компонентҳои дигар, ба монанди Apache ва MySQL, барои таҳия ва
санҷишҳои маҳаллӣ кӯмак мекунад, ки барои кор бо базаҳои дода ва
серверҳои веб муҳим аст \[19 c. 30, 102\].

**2.1.6. Системаи идоракунии версия ва тақсимоти вазифаҳо: Git ва
GitHub**

Git система барои назорати версияҳо мебошад, ки имкон медиҳад тағйироти
кодро пайгирӣ намоем ва версияҳои гуногуни лоиҳаро идора кунем. Git ба
гурӯҳи таҳиягарон имкон медиҳад самаранок кор кунанд, таърихи тағйиротро
нигоҳ доранд ва ба осонӣ тағйирот ворид кунанд \[20, c 7, 45\].

GitHub барои гурӯҳҳои таҳиягарон як платформаи бисёрфунксионалӣ мебошад,
ки барои назорати версия, ҳамоҳангсозии кор, ва гурӯҳбандии вазифаҳо
хеле мувофиқ аст. Бо ёрии GitHub, мо метавонем вазифаҳоро байни аъзои
гурӯҳ тақсим кунем ва як равандҳои ҳамоҳангсозӣ ва ҳамкориро осон
намоем. Ҳар як аъзои гурӯҳ метавонад бо истифода аз **branches**
(шохаҳо) барои таҳрир ва коркарди вазифаҳояшон мустақилона кор кунад, бе
он ки ба асоси код таъсир расонад.

Ҳамчунин, GitHub имконият медиҳад, ки **issues** (масъалаҳо) барои
муайян кардани вазифаҳо, шикастҳо ва такмилдиҳии система истифода
шаванд. Масъалаҳо метавонанд барои таъйин кардани вазифаҳо, нишонагузорӣ
ва пайгирии кори гурӯҳ истифода шаванд, ки ин тамоми равандро самаранок
ва сабук мекунад.

Бо ёрии **pull requests** (талаботҳои пайваст) метавон пайгирӣ кард, ки
тағиротҳои воридшудаи аъзои гурӯҳ чӣ гуна ба коди асосӣ таъсир
мерасонанд. Тағирот пеш аз ворид кардан ба шохаи асосӣ аз тарафи дигар
аъзои гурӯҳ санҷида ва арзёбӣ карда мешаванд, ки ин бартараф кардани
хатогиҳо ва беҳтар кардани сифати кодро таъмин мекунад.

GitHub инчунин бо **actions** (ҳодисаҳо) ва **CI/CD пайвандҳо** имкон
медиҳад, ки равандҳои автоматизатсионӣ, мисли санҷиши автоматӣ ва
пайвастшавии доимии барнома, барои таъмини сифати код ва кори бехатари
лоиҳа амалӣ карда шаванд.

Ин имкониятҳо ва функсияҳои GitHub барои гурӯҳҳои таҳиягарон кӯмак
мекунанд, ки кор мунтазам, равшан ва самаранок анҷом дода шавад.

Figma -- ин як барномаи тарроҳии графикӣ ва прототипсозӣ мебошад, ки ба
корбарон имкон медиҳад, ки дар муҳити онлайн тарҳҳо эҷод ва ҳамкорӣ
кунанд.

Он барои тарроҳони веб ва UI/UX бисёр муфид аст, зеро имкон медиҳад, ки
чанд нафар ҳамзамон дар як лоиҳа кор кунанд. Figma воситаҳои гуногун
барои кашидани интерфейс, сохтани компонентҳо ва прототипҳои
интерактивиро пешниҳод мекунад.

Афзалиятҳои Figma:

-   кор дар муҳити онлайн бе ниёз ба насб кардани барнома;

-   дастрасӣ ба файлҳо аз ҳар ҷо;

-   ҳамкории дастаӣ дар вақти воқеӣ;

-   захираи худкор ва таърихи версияҳо.

Figma барои тарроҳон, барномасозон ва ҳатто мизоҷоне, ки мехоҳанд тарҳро
баррасӣ кунанд, воситаи қулай мебошад.

Интихоби воситаҳои таҳия барои барнома, воситаҳои мувофиқ барои таҳияи
барнома баррасӣ шуданд. Интихоби дурусти технологияҳо ва абзорҳои таҳия
нақши муҳим дар самаранокӣ, устуворӣ ва васеъшавандагии барнома дорад.

Бо назардошти талабот ва хусусиятҳои лоиҳа, истифодаи забонҳои
барномасозӣ, муҳити рушд, системаҳои идоракунии базаи додаҳо ва дигар
воситаҳои зарурӣ муайян карда шуд. Ҳар як интихоб бо назардошти
самаранокии иҷроиш, осонии таҳия ва ҳамоҳангӣ бо дигар ҷузъҳои система
асоснок карда шуд.

Ҳамин тариқ, интихоби дуруст ва мутобиқ ба талаботи лоиҳа имконият
медиҳад, ки барнома бо сифати баланд таҳия шуда, дар муҳити корӣ
самаранок фаъолият намояд.

###  **Тарҳрезии модели ташкили система ва назорати давомоти толибилм дар мактаби миёна**

Раванди тарҳрезӣ қисмати асосии коркарди воситаи барномавӣ ба шумор
меравад. Ин қадам ба коркардчиён ва фармоишгар имконияти ҳамфикр будан
ва дар оғози коркард дида баромадани паҳлӯҳои системаи коркардмешударо
фароҳам меорад. Аз ин рӯ, қарор карда шуд, то системаи назорати давомоти
толибилм дар мактаби миёна сараввал тариқи модел тарҳрезӣ гардад.

Барои модели диаграммаҳои системаи назорати давомоти хонандагон дар
мактаби миёна якчанд диаграммаҳои калидӣ барои визуализатсияи меъмории
системаро пешниҳод кардан мумкин аст:

Диаграммаи ҳолати истифодабарӣ (Use Case) як намуди диаграмма дар забони
ягонаи моделсозӣ (UML) мебошад, ки барои намоиш додани талаботи
функсионалии система ва ҳамкории корбарон (актерҳо) бо система истифода
мешавад. Он тавсиф мекунад, ки кадом функсияҳоро (ҳолатҳои истифода)
система бояд иҷро кунад ва корбарон (актерҳо) бо ин функсияҳо чӣ гуна
ҳамкорӣ хоҳанд кард \[20\].

Ин диаграмма барои омодасозии система, ки аввал бояд кадом нақшҳо чи
корҳоро бояд иҷро кунанд, истифода мешавад.(ниг. ба расми 1)

**Расми 2. Диаграммаи ҳолати истифодабарии система**

Дар ин ҷо нишон дода шудааст, ки омӯзгор метавонад ба система дохил шуда
барои гузоштани баҳо ва бақайдгирии давомоти хонандагон аввал синфи дарс
медодаро интихоб намуда баъд хонандаро интихоб намояд. Волидон
метавонанд аввал фарзандони худро, ки дар мактаб таҳсил мекунанд,
интихоб намуда пешрафт ва давомоти онҳоро назорат кунанд. Хонандагон
метавонанд дар система пешрафт ва давомоти худро бинанд.

Диаграммаи Entity-Relationship (ER) як намоиши визуалии сохтори пойгоҳи
додаҳо мебошад, ки дар тарҳрезӣ ва фаҳмидани муносибатҳои байни
объектҳои гуногун дар пойгоҳи додаҳо кӯмак мекунад. Он дар моделсозии
пойгоҳи додаҳо барои сохтори додаҳо пеш аз татбиқ дар системаи
идоракунии пойгоҳи додаҳо (RDBMS) васеъ истифода мешавад \[21\].

**Расми 3. Диаграмаи алоқаи байни ҷадвалҳои базаи маълумот**

Чи хеле, ки дар расм нишон дода шудааст диаграммаи Entity-Relationship
(ER) як намоиши визуалии сохтори пойгоҳи додаҳо мебошад. Дар он аввал
ҷадвалҳои манбаи додаҳо сохта шуда баъд барои алоқаи байни ҷадвалҳо аз
рӯи раванди тиҷоратӣ калиди беруна дода мешавад. Дар ин диаграмма як
ҷадвал ба дигараш чи гуна додугирифт мекунад, нишон дода шудааст. Бо
ёрии аломатҳо яке бо дигаре чи тавр пайваст аст, масалан ҷадвали Синфҳо
ба ҷадвали Вазифаи хонагӣ алоқаи як ба бисёр дорад. Ин маънои онро
дорад, ки як синф метавонад якчанд вазифаҳои хонагиро дошта бошад.

Диаграммаи BPMN (Модели равандҳои тиҷоратӣ ва қайд) диаграммаест, ки
барои моделсозии равандҳои тиҷорат истифода мешавад. BPMN ба шумо имкон
медиҳад, ки равандҳо, нақшҳо, қадамҳо ва таъсири мутақобила байни
иштирокчиёни системаҳои тиҷоратро тавсиф кунед. Дар заминаи системаи
ҳузури хонандагони мактаб, BPMN метавонад барои нишон додани равандҳои
марбут ба сабти давомот, огоҳиҳои волидайн ва гузоришдиҳӣ истифода шавад
\[22\].

**Расми 4. Схемаи BPMN барои раванди бақайдгирии давомот**

Дар расми зикр гардида нишон дода шудааст, ки чи гуна омӯзгор барои
бақайдгирии хонандагон равандҳоро иҷро мекунад. Омӯзгор ба система логин
ва рамзи худро дохил намуда баъд ба қисми давомот дохил мешавад. Дар
саҳифаи давомот рӯйхати синфҳое, ки омӯзгор дарс медиҳад, оварда
мешавад. Рӯйхати синфҳо ба система аз манбаи маълумот гирифта мешавад.
Баъд синфро интихоб мекунад, ки дар натиҷа рӯйхати хонандагони ин синф
дар намуди ҷадвал оварда мешавад. Баъд аз рӯи хонандагони ҳузур дошта ё
ҳузур надошта тугмаҳои ҳаст нест пахш карда мешавад. Ҳангоми фаромӯш
кардани яке аз хонандагон система ба истифодабаранда хабар медиҳад. Агар
ягон хатоги вуҷуд надошта бошад ин давомот ба манбаи маълумоти система
сабт карда мешавад.

Дар ҷараёни тарҳрезии системаи идоракунӣ ва назорати давомоти хонандагон
дар мактаби миёна диаграммаи
BPMN ([англ.](https://ru.wikipedia.org/wiki/Английский_язык) Business
Process Model and Notation, нишондод ва модели раванди бизнес),
диаграммаи "Объект - Пайвастшавӣ" (ERD) ва диаграммаи ҳолатҳои истифода
(Use Case Diagram) таҳия ва таҳлил карда шуданд.

1\. Диаграммаи BPMN равандҳои асосии системаро, аз ҷумла бақайдгирӣ ва
ворид шудан ба система, идоракунии баҳогузорӣ ва давомотро нишон дод.
Истифодаи зерравандҳо (sub-processes) сохтори диаграммаро содда карда,
равандҳоро равшан ва фаҳмо сохт.

2\. Диаграммаи " Объект - Пайвастшавӣ" (ERD) барои тарҳрезии базаи
додаҳо кумак намуд, ки дар он моҳиятҳо ба монанди Хонанда, Омӯзгор,
Волидайн, Администратор, Баҳогузорӣ, Вазифаи хонагӣ, Давомот, Синф, Фан
ва муносибати байни онҳо муайян карда шуданд. Ин ба таҳкими сохтори
додаҳо ва мутобиқати иттилоот мусоидат намуд.

3\. Диаграммаи ҳолатҳои истифода (Use Case Diagram) нақшҳои асосии
система ва функсияҳои дастрас барои ҳар як истифодабаранда (маъмур,
омӯзгор, хонанда, волидайн) нишон дод. Ин диаграмма ҳудуди система ва
сенарияҳои асосии корбаронро муайян намуда, ҳуқуқҳои ҳар як нақшро
равшан сохт.

Истифодаи ин моделҳо ба тартиб додани сохтори система, ошкор намудани
хатогиҳо дар марҳилаи аввал ва беҳтар кардани фаҳмиши мантиқии кори
система мусоидат намуд. Дар асоси ин диаграммаҳо, метавон барномаи
самаранок ва устувори системаеро анҷом дод, ки барои корбарон қулай ва
муассир хоҳад буд.

### **Омодасозии система ва муҳофизати он**

Барои омода сохтани система асбобҳои зарурӣ интихоб гардидааст, ки барои
самаранок истифода бурдани система хизмат мерасонад. Барои нигоҳ доштани
маълумотҳои система ва коркарди маълумотҳои он аз забони дархостгузории
MySQL истифода бурда шудааст.

Қисми амнияти дар ҳар системае, ки дар шабака кор мекунад ин ҷузъи
асосии барнома ба ҳисоб меравад. Инҳо асоситарин чораҳои бехатарӣ барои
ҳимояи маълумот ва пешгирии дастрасии ғайриқонунӣ мебошанд:

1.  **Иҷозатдиҳӣ**

Барои маҳдуд кардани дастрасӣ ба система ҳамаи истифодабарандагон бояд
паролҳои мураккаб(ҳадди ақал 8 рамз бо ҳарфҳои калон, хурд, рақамҳо ва
аломатҳои махсус) дошта бошанад. Ҳар як корбар танҳо ба маълумоте, ки
барои ӯ лозим аст, дастрасӣ дорад (хонандагон ба маълумоти шахсии худ,
муаллимон ба маълумоти синф, маъмурон ба тамоми система).

2.  **Муҳофизати маълумот**

Барои пешгирии дуздии маълумот паролҳо бо истифодаи Bcrypt шифргузори
карда мешавад. Ин имкон медиҳад, ки ҳатто дар ҳолати кушода шудан низ
шикаста нашаванд.

3.  **Захира ва барқарорсозии маълумот**

Барои пешгирии гум шудани маълумот нусхаи эҳтиётии автоматӣ (Backup)
истифода мешавад, ки дар он маълумот ҳар рӯз ба сервери дигар
нусхабардорӣ мешавад.

Барои таҳия намудани саҳифаҳои барнома макети он сохта шудааст, ки ин
имкон медиҳад, пешаки чӣ тавр саҳифаҳои система зоҳир мегардад, дида
шавад.

## **ҶОРИСОЗӢ, ТАҲЛИЛИ ИҚТИСОДӢ ВА САМАРАНОКИИ СИСТЕМА**

Арзёбии низоми умумӣ, иқтисоднокӣ ва самаранокии он марҳилаи муҳими ҷорӣ
намудани системаи автоматишудаи назорати давомоти толибилм дар мактаби
миёна мебошад. Таҳия ва татбиқи чунин системаҳо на танҳо таҳлили
техникӣ, балки асосноккунии иқтисодиро низ талаб мекунад, зеро ҳама гуна
тағйирот дар муассисаҳои таълимӣ бояд манфиатовар ва боиси коҳиш додани
хароҷот ва баланд бардоштани сифати идоракунии ҳузур бошанд.

Дар ин бахш нишондиҳандаҳои асосии самаранокии система, таъсири он ба
раванди таълим ва асоснокии иқтисодии татбиқи он баррасӣ карда мешаванд.
Ба коҳиш додани хароҷоти маъмурӣ барои назорати давомот, беҳтар намудани
таҳлили маълумот ва баланд бардоштани интизоми хонандагон таваҷҷуҳи
махсус дода мешавад. Таҳлили муқоисавии усулҳои анъанавӣ ва
автоматишудаи назорати давомоти толибилм гузаронида шуда, сатҳи
самаранокии лоиҳа ва манфиатҳои дарозмуддати он муайян карда мешаванд.

                        \$subject\[\'teacher_id\'\],

                        \$day,

                        \$time_slot\[0\],

                        \$time_slot\[1\]

                    \]);

Ин код барои тақсимоти дарсҳо дар ҷадвали дарсӣ истифода мешавад. Он
мекӯшад, ки ба ҳар як фан дар асоси талабот вақт ва рӯз ҷудо кунад, то
ки дар раванди таҳсилот низоъҳо (конфликт) ба миён наоянд. Рӯз ва вақти
дарсӣ бо усули тасодуфӣ интихоб карда мешавад. Агар фан ва муаллим дар
ҳамин вақт ва рӯз дар дигар синфхонаҳо дарс надошта бошанд, код ба
қадами навбатӣ мегузарад. Ин код ба базаи додаҳо саволи SQL (query)
мефиристад, то фаҳмад, ки чунин дарс аллакай вуҷуд дорад ё не. Агар дарс
мавҷуд набошад, он метавонад ба ҷадвал илова карда шавад.

Дар расми 31 саҳифаи генератори ҷадвали дарсӣ оварда шудааст.

**Расми 31. Саҳифаи генератори ҷадвали дарсӣ**

Дар ин саҳифа нишон дода шудааст, ки барои кор кардани ин аввал ҳамаи
синфҳо, фанҳо дохил карда шудаанд. Омӯзгорон ба синфҳо ва синфҳо ба
фанҳо вобаста карда шудаанд ва миқдори соатҳои ҳафтаина барои ҳар як фан
муайян карда шудааст. Баъди ҳамаи ин амалҳо метавонанд ҷадвали дарсро
генератсия кунанд. Дар ин боз имконияти зарурӣ мавҷуд аст, ки пеш аз
генератсияи ҷадвали нав, нусхаи пойгоҳи додаҳо сабт карда мешавад.

Системаи таҳияшуда ба таври назаррас раванди идоракунии ҷараёни таълимро
осон ва самаранок мегардонад. Бо илова кардани имкониятҳои нав, аз ҷумла
генератори ҷадвали дарсӣ, идоракунии сабабҳои набудани хонандагон ва
огоҳиномаи автоматикӣ, сатҳи назорат ва шаффофият дар муассисаҳои
таълимӣ баланд бардошта мешавад.

Инноватсияҳои воридгардида имкон медиҳанд, ки ҳам омӯзгорон ва ҳам
волидайн дар вақти воқеӣ маълумоти лозимаро дастрас намоянд, ки ин барои
беҳтар намудани иштироки хонандагон дар раванди таълим мусоидат мекунад.

Бо рушди минбаъдаи система, имконияти васеъ намудани функсионалӣ ва
мутобиқ кардани он ба талаботи нав пайдо мешавад. Истифодаи чунин ҳалли
рақамӣ дар муассисаҳои таълимӣ қадами муҳим ба сӯи рақамикунонии пурраи
идоракунии таҳсилот дар Тоҷикистон хоҳад буд.

### **Самаранокии иқтисодии система**

Низоми автоматикунонидашудаи назорати давомоти хонандагон барои баланд
бардоштани интизом, осон намудани бақайдгирии ҳозиршавӣ ва кам кардани
хароҷоти маъмурӣ пешбинӣ шудааст. Ворид кардани чунин система
маблағгузории ибтидоиро талаб мекунад, аммо дар вақти дарозмуддат он
фоидаи иқтисодии назаррас меорад. Дар ин бахш хароҷот, бартариҳо ва
камбудиҳои система, инчунин мӯҳлати баргардонидани маблағ таҳлил карда
мешаванд.

Хароҷоти система ба якдафъаина (барои татбиқ ва хариди таҷҳизот) ва
давомдор (барои дастгирӣ, навсозӣ ва нигоҳдорӣ) тақсим мешавад.

**Афзалиятҳо ва камбудиҳои система**

Афзалиятҳо

1.  Сарфаи вақти омӯзгорон -- бақайдгирии худкор вақти зиёдро сарфа
    мекунад.

2.  Кам кардани хатогиҳои инсонӣ -- сабти ҳозиршавӣ ба таври автоматикӣ
    анҷом мешавад.

3.  Огоҳии фаврии волидайн -- тавассути хабарҳои системавӣ аз набудани
    фарзандашон бохабар мешаванд.

4.  Баланд бардоштани интизоми хонандагон -- назорат аз ҷониби мактаб ва
    волидайн коэффисиенти ҳозиршавиро зиёд мекунад.

5.  Имконияти васеъ ва мутобиқшавӣ -- система метавонад дар якчанд
    мактаб истифода шавад.

6.  Анализ ва ҳисоботҳо -- маъмурият метавонад маълумоти ҳозиршавиро дар
    вақти воқеӣ пайгирӣ кунад.

Камбудиҳо

1.  Арзиши баланди ибтидоӣ -- таҳияи нармафзор ва хариди таҷҳизот
    маблағи зиёдро талаб мекунад.

2.  Лозим будани омӯзиши кормандон -- омӯзгорон ва маъмурон бояд
    истифодаи системаро омӯзанд.

3.  Вобастагӣ ба мушкилоти техникӣ -- дар ҳолати қатъ шудани интернет ё
    вайрон шудани сервер маълумот дастнорас мешавад.

4.  Хароҷоти иловагӣ барои нигоҳдорӣ -- навсозии нармафзор ва таҷҳизот
    хароҷоти давомдор дорад.

**Арзёбии самаранокии иқтисодӣ ва мӯҳлати баргардонидани маблағ**

Система тавассути кам кардани хароҷоти маъмурӣ ва баланд бардоштани
ҳозиршавии хонандагон сарфаҷӯиро таъмин мекунад.

Сарфаи хароҷоти маъмурӣ

-   коҳиши хароҷот барои ҳуҷҷатгузории коғазӣ -- то 5 000 сомонӣ дар як
    сол;

-   сарфаи вақти омӯзгорон -- то 10 000 сомонӣ дар як сол;

-   самаранокии кори маъмурият -- то 7 000 сомонӣ дар як сол.

Баланд бардоштани ҳозиршавӣ

Маърифати хубтар ва иҷрои беҳтари дарсҳо -- кам шудани миқдори
донишомӯзони нофаъол, пешгирии хатари хориҷшавии хонандагон -- ки
метавонад маблағгузории мактабро муҳофизат кунад.

Ҳисобкунии мӯҳлати баргардонидани маблағ

Агар мактаб 20 000 -- 40 000 сомонӣ сармоягузорӣ кунад ва сарфаҷӯии
солона 12 000 -- 20 000 сомонӣ бошад, система дар муддати 1 -- 2 сол
хароҷоти худро ҷуброн мекунад.

Низоми назорати ҳозиршавии хонандагон инвеститсияи муфид барои
муассисаҳои таълимӣ мебошад, зеро он:

-   Хароҷоти маъмуриро кам мекунад.

-   Интизоми хонандагонро беҳтар менамояд.

-   Назорати раванди таълимро осон ва шаффоф мегардонад.

Гарчанде ки хароҷоти ибтидоии система баланд аст, он дар давоми 1--2 сол
пурра ҷуброн мешавад ва пас аз ин фоидаи тоза ба бор меорад.

Дар ин бахш, ҷорӣ намудани низоми назорати давомоти хонандагон таҳлил
карда шуд, хатогиҳо ислоҳ гардид, имкониятҳои иловагӣ ва самаранокии
иқтисодии он баррасӣ карда шуд.

Санҷиши система нишон дод, ки он ба талаботи муассисаҳои таълимӣ ҷавобгӯ
буда, боэътимод ва самаранок фаъолият мекунад. Ислоҳи хатогиҳо ва такмил
додани функсияҳо имкон дод, ки кори система бе маҳдудият ва бо дақиқии
баланд иҷро шавад.

Имкониятҳои иловагии система, аз ҷумла огоҳиномаи автоматикӣ барои
волидайн, ҳисоботи муфассал барои маъмурият ва мутобиқшавӣ бо дигар
хидматҳо, самаранокии онро зиёд намуд.

Таҳлили иқтисодӣ нишон дод, ки новобаста аз хароҷоти ибтидоӣ, система
дар давоми 1-2 сол хароҷоти худро ҷуброн карда, дарозмуддат ба сарфаи
маблағ ва беҳтар шудани интизоми хонандагон мусоидат менамояд.

Бо назардошти ҳамаи омилҳо, метавон хулоса кард, ки ҷорӣ намудани ин
система дар мактабҳо як қадами муҳим ба сӯи автоматикунонии идоракунии
раванди таълим ва баланд бардоштани самаранокии фаъолияти таълимӣ
мебошад.

## **ХУЛОСА** {#хулоса .unnumbered}

Таҳияи системаҳои назорати давомоти толибилм дар мактаби миёна яке аз
самтҳои муҳими рақамикунонии соҳаи маориф ба ҳисоб меравад. Дар доираи
ин кор системае таҳия карда шуд, ки имконият медиҳад назорати давомоти
хонандагон дар мактабҳои миёна ба таври автоматӣ ва муассир ба роҳ монда
шавад.

Системаи таҳияшуда бо истифода аз технологияҳои муосири веб --- HTML,
CSS, JavaScript, PHP ва MySQL амалӣ карда шудааст. Интерфейси он барои
омӯзгорон, маъмурият ва волидайн фаҳмо ва қулай буда, имконият медиҳад,
ки ҳар як иштирокчии раванди таълим ба маълумоти лозима дастрасии фаврӣ
дошта бошад. Омӯзгорон метавонанд давомотро сабт намоянд, волидайн
маълумотро дар бораи ҳозирии фарзандони худ бубинанд ва маъмурият
ҳисоботҳои таҳлилӣ гирад.

Дар марҳилаи таҳияи лоиҳа таҳлили ҳолати ҷории сабти ҳозирӣ дар мактабҳо
гузаронида шуда, мушкилоти асосӣ муайян карда шуданд. Ин мушкилот ба
монанди хатогиҳои зиёд, сарфи зиёди вақт ва набудани шаффофият боиси
коҳиши самаранокии идоракунии таълим мегарданд. Бо татбиқи системаи нав,
ҳамаи ин нуқсонҳо бартараф карда мешаванд.

Система дорои имкониятҳои зерин мебошад:

-   бақайдгирии автоматии ҳозирӣ;

-   фиристодани огоҳиномаҳо ба волидайн дар бораи дер омадан ё ғоиб
    будани фарзанд;

-   таҳлили оморӣ ва ҳисоботдиҳӣ барои маъмурият;

-   ҳифзи маълумоти шахсӣ ва дастрасии маҳдуд барои истифодабарандагон
    вобаста ба нақш.

Истифодаи платформаи GitHub барои назорати версияҳо ва муҳити таҳияи VS
Code барои рушди самараноки барнома имкон доданд, ки лоиҳа бо сатҳи
баланди касбӣ таҳия гардад. Илова бар ин, барнома дар муҳити саросарӣ бо
истифодаи системаи дастраскунии хостингҳо ва серверҳои Beget омода ва
санҷида шудааст, ки имкон медиҳад системаро дар ҳар ҷо истифода бурд.

Таҳлили иқтисодӣ нишон дод, ки система на танҳо самаранок, балки аз
ҷиҳати хароҷот низ муфид аст. Истифодаи он ба коҳиши хароҷоти иловагӣ,
кам кардани кори дастӣ, баланд бардоштани интизоми хонандагон ва беҳтар
кардани ҳамкории мактаб бо волидайн мусоидат мекунад. Бо ёрии ин
система, омӯзгорон вақти худро бештар ба таълим бахшида метавонанд ва
маъмурият дар қабул кардани қарорҳои муҳим маълумоти дақиқ ва саривақтӣ
дар ихтиёр хоҳад дошт.

истемаи таҳияшуда ба талаботи мактабҳои муосир ҷавобгӯ буда, метавонад
ҳамчун намуна барои дигар муассисаҳои таълимӣ истифода шавад. Ҷорӣ
намудани чунин система як қадами муҳим дар роҳи рақамикунонии маориф ба
ҳисоб меравад ва ба баланд бардоштани самаранокии таҳсил мусоидат
менамояд.
